# Project Overview

This project is a personal portfolio website built using **HTML, CSS, and JavaScript**.  
It presents my profile, projects, skills, education, and contact information in a clean, dark-themed layout.

The website is deployed using **AWS**, and the source code is maintained on **GitHub**.

---

##  Technologies Used

- HTML5  
- CSS3  
- JavaScript  
- Git & GitHub  
- AWS (Static Website Hosting)

---

##  Deployment Process

### Step 1: Project Preparation

- All files are organized properly  
- `index.html` links to external CSS and JavaScript files  
- The website is tested locally in a browser  

### Step 2: AWS Deployment (Static Website)

- Create an S3 bucket with a unique name  
- Enable Static Website Hosting  
- Upload all project files to the bucket  
- Set `index.html` as the index document  
- Configure bucket permissions to allow public access  
- Access the website using the AWS-generated endpoint URL
